Q1

Data is a collection of features or information, usually numbers, by observation. Technically, a data is a set of values for a qualitative or quantitative variable relating to one or more people or objects, while data (the singular number of data) is a single value for a single variable. Information can be thought of as a solution to uncertainty; It is this answer to the question "what is an entity?" that defines the essence of an entity and the essence of its features. The concept of information has different meanings in different contexts. Although the terms "data" and "information" are often used interchangeably, these terms have different meanings. I think data is just a unit of information.
